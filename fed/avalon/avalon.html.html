    avalon拥有4种填空数据的功能。

    前两种是绑定属性，分别是文本绑定与HTML绑定，它们会清空原元素的内部再进行内部填充。后两种是插值表达式，{{text}}是对某一个文本节点的nodeValue进行修改，因此不会影响兄弟节点。 {{text|html}}添加了一个叫html的过滤器，会将文本绑定变成HTML绑定，将目标字符串转换为文档碎片，然后与所在的文本节点进行replaceChild操作，因此也不影响兄弟节点。 

            <script>
           
         avalon.define("test", function(vm) {
              vm.text = "<b> 1111  </b>"
         })

        </script>
        <div ms-controller="test">
            <div><em>用于测试是否被测除</em>xxxx{{text}}yyyy</div>
            <div><em>用于测试是否被测除</em>xxxx{{text|html}}yyyy</div>
            <div ms-text="text"><em>用于测试是否被测除</em>xxxx yyyy</div>
            <div ms-html="text"><em>用于测试是否被测除</em>xxxx yyyy</div>
        </div>
    用于测试是否被测除xxxx{{text}}yyyy
    用于测试是否被测除xxxx{{text|html}}yyyy
    用于测试是否被测除xxxx yyyy
    用于测试是否被测除xxxx yyyy

    此外，我们还可以通过配置方式，更改插值表达式的界定符，因为{{}}可能被其他框架的模板所占用。建议界定符的长度大于1，不要设置为>>这样的位操作符。比如在DOMReady之前，我们调用如下语句： 

                avalon.config({
               interpolate: ["<%", "%>"] //  要求openTag 不等于closeTag就能配置成功
            })
