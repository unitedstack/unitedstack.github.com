concept.method

描述：

ViewModel可以混杂许多方法，作用如同controller的action一样，这样我们就可以把复杂的逻辑分离出页面。 

         <p>请输入名字: <input ms-model="nick" /></p>
         <p>你的头衔是{{getName()}}！！！</p>   
         <p>{{nick}}</p>     
            
require("more/avalon, ready", function($) {
   $.avalon.define(function(a){
       a.nick = "ruby";
       a.getName = function(){//method
          return a.nick +"将军"
       } 
   })
   $.avalon.scan()
});
