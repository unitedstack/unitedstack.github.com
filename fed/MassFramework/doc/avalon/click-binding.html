click-binding

描述：

点击绑定是一个最常用的事件绑定，其实它是内部调用了event绑定，语法为ms-click="fn" 

        <div>你已经点击了{{numberOfClicks}} 次</div>
        <button ms-click="registerClick" ms-disabled="hasClickedTooManyTimes">点我</button>
        <div ms-visible="hasClickedTooManyTimes">
            你已经点击了许多次,不能再说了,请<button ms-click="resetClicks">重置</button>
        </div>
            
            require("more/avalon, ready", function($) {
                 $.avalon.define(function(a) {
                    a.numberOfClicks = 0;
                    a.registerClick = function() {
                        a.numberOfClicks++;
                    };
                    a.resetClicks = function() {
                        a.numberOfClicks = 0;
                    };
                    a.hasClickedTooManyTimes = {
                        get: function() {
                            return this.numberOfClicks >= 5;
                        }
                    }
                });
                $.avalon.scan();
            });
